# Sistema de Debida Diligencia Anticorrupción (SDDA)

Proyecto académico desarrollado para la asignatura Ingeniería de Software IV.

## Tecnologías

- React
- Node.js
- Express
- PostgreSQL
- SonarCloud
- Jest

## Equipo

- Arquitecto de Soluciones
- Desarrollador Frontend
- Desarrollador Backend

## Objetivo

Automatizar el proceso de Debida Diligencia Anticorrupción conforme a la Ley 23 de 2015 y la ISO 37001.
